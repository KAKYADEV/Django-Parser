# Django-Parser
A site for parsing any user-requested site

Основная рабочая идея - пользователь вводит URL сайта и тип информации, которую хочет спарсить, встроенные функции отрабатывают запрос и данные попадают в базу, из которой пользователь их может получить в удобном для себя формате.
